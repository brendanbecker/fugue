# FEAT-083: Protocol Generalization: Generic Widget System

**Priority**: P1
**Component**: fugue-protocol
**Type**: enhancement
**Estimated Effort**: medium
**Business Value**: medium

## Overview

Remove hardcoded integration structs (e.g., BeadsTask) from the core binary protocol. Implement a generic 'Widget' or 'Extension' message type capable of carrying arbitrary JSON payloads for client-side rendering. This decouples the multiplexer from specific tools.
